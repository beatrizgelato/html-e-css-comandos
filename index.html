Principais Comandos para Criar um Site

Comandos Html

HTML é uma linguagem de marcação utilizada para definir a estrutura de páginas da web. Os principais elementos incluem:

  • <html></html>: Determina o começo e fim do html
  • <head></head>: Inclui metadados, como o título da página e referências a arquivos CSS.
  • <body></body>: Contém todo o conteúdo visível da página na web.

Comandos Css

CSS é usado para estilizar o conteúdo HTML. Alguns comandos básicos incluem:

  • color: Define a cor do texto.
  • font-size: Define o tamanho da fonte.
  • font-family: Define a fonte a ser usada.
